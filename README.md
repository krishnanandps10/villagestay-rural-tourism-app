Overview

Athithya (VillageStay) is a mobile application built to promote rural tourism by showcasing curated village stays and cultural experiences.

The platform connects tourists, merchants, NGOs, and admins through a unified system.

Key Features

Flutter-based mobile frontend
FastAPI backend APIs
Secure authentication using Supabase Auth
Role-based access (Tourist, Merchant, NGO, Admin)
Booking and submission workflows
Media management for listings

Tech Stack

Flutter
FastAPI
Supabase Auth
Supabase Database & Storage
Python



Outcome

Created a scalable rural tourism platform enabling real-time interaction between stakeholders while promoting sustainable tourism.
